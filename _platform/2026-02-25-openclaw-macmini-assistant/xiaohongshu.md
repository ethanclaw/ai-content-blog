1️⃣ 消息统一管理 - 微信/Telegram/飞书消息一个入口，再也不用来回切换了！

## 为什么选 Mac Mini？💡

• 功耗低 - 只有 30W，待机几乎没声音 🔇
• 一直开着不心疼 - 24小时运行没问题
• M4 芯片 AI 能力很强 - 好几年不过时

---

## 安装超简单！🚀

```
npm install -g openclaw
openclaw init
openclaw gateway start
```

配置 Telegram/Discord 通道，接入 LLM（推荐 OpenRouter，性价比高）就能用啦～

---

## 怎么让它更懂我？🧠

写 3 个文件就行：

• SOUL.md → 告诉它我是谁、性格啥样
• USER.md → 我的名字、偏好、时区
• MEMORY.md → 重要的上下文、决策记录

时间长了它就越来越懂我啦，像个小伙伴一样！👭
